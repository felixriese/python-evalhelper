# python-evalhelper

Python Abschlussprojekt SoSe 2017 - Evaluationshelfer

## System requirements

You need Python 3.x installed with the following packages:

* PIL
* numpy
* scipy
* matplotlib and seaborn (maybe later)

## Usage

```bash
python3 evalhelper_mail.py
```
